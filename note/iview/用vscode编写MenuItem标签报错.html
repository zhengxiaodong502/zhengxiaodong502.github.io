iview用vscode编写MenuItem标签报错

Sun.zheng 2018-07-09 iview

这是vetur中eslint的问题 在vscode菜单中，

window: 文件->首选项->设置

Mac: code -> 首选项->设置

输入eslint ,打开settings.json

找到 “vetur.validation.template”: true 将其改为false
